# Google OAuth PoC

Google OAuth öğrenmek için yaptığım çalışma.

## Kurulum

1. Repo'yu klonlayın
```bash
git clone https://github.com/MetaMsa/Google-OAuth-PoC.git
cd Google-OAuth-PoC
```
2. Kendi env dosyanızı oluşturun
3. Docker compose ile projeyi ayağa kaldırın
```bash
docker compose up -d --build
```
